NCOG SDK Browser Example

Setup Instructions

To use the NCOG SDK in a browser:

  1. Include the Go WebAssembly runtime: <script src="wasm_exec.js"></script>
  2. Import the SDK: import { loadWasm } from '../dist/index.browser.esm.js'
  3. Initialize the WASM module: const mlkem = await loadWasm()
